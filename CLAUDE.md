# Geoforce.ai Landing Page

## Tech Stack
- Next.js 16 (App Router, Turbopack)
- Tailwind CSS v4
- React Three Fiber + Three.js (3D hero)
- tsparticles (ember particles)
- Framer Motion (scroll animations)

## GSAP Skills
When implementing GSAP animations, reference the skills in `skills/`:
- `skills/gsap-core/SKILL.md` — Core API, tweens, easing, stagger
- `skills/gsap-scrolltrigger/SKILL.md` — Scroll-driven animations
- `skills/gsap-timeline/SKILL.md` — Sequencing, timelines
- `skills/gsap-react/SKILL.md` — React integration with useGSAP()
- `skills/gsap-plugins/SKILL.md` — Flip, Draggable, SplitText, MorphSVG
- `skills/gsap-performance/SKILL.md` — Performance best practices
- `skills/gsap-frameworks/SKILL.md` — Framework integration patterns
- `skills/gsap-utils/SKILL.md` — Helper utilities
